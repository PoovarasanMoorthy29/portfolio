# 🎯 START HERE - Portfolio Setup Guide

Welcome to your professional portfolio! This guide will get you started in minutes.

## ⚡ Quick Start (5 Minutes)

### Step 1: Add Your Images (2 min)
Create these image files and save them to the `assets/` folder:
- `profile.jpg` - Your professional photo (300x300px)
- `project1.jpg`, `project2.jpg`, `project3.jpg` - Project screenshots
- `resume.pdf` - Your resume

**No images ready?** Use [Unsplash](https://unsplash.com) for free photos, or generate placeholders at [Placeholder.com](https://placeholder.com).

### Step 2: Update Your Information (2 min)
Edit `index.html` and find-and-replace:
- `poovarasan@example.com` → your email
- `+919876543210` → your phone
- `https://github.com` → your GitHub
- `https://linkedin.com` → your LinkedIn
- `Chennai, Tamil Nadu, India` → your city

### Step 3: Open in Browser (1 min)
- Double-click `index.html`, OR
- Use Live Server in VS Code

**Done!** Your portfolio is ready to view and customize.

## 🎬 Common Tasks

### Add Your Profile Photo
1. Save a 300x300px image as `profile.jpg`
2. Place in `assets/` folder
3. Reload page
4. Done! Your photo appears in hero section

### Update Your Email
1. Open `index.html`
2. Find all instances of `poovarasan@example.com`
3. Replace with your email
4. Save file
5. Refresh browser

### Change Colors
1. Open `css/style.css`
2. Find `:root { ... }`
3. Change `--accent-primary: #00d4ff;`
4. Save file
5. Refresh browser

### Add a New Project
1. Open `index.html`
2. Find the projects section
3. Copy one project-card div
4. Fill in your project details
5. Add project image to `assets/`
6. Save and refresh

### Test on Mobile
1. Open DevTools (F12)
2. Click Device Toolbar (Ctrl+Shift+M)
3. Select mobile device
4. Test all features
5. Close DevTools (F12)

### Find and Fix Issues
1. Open DevTools (F12)
2. Go to Console tab
3. Look for red error messages
4. Click error to get details
5. Check [TROUBLESHOOTING.md](TROUBLESHOOTING.md)

## ❓ Frequently Asked Questions

**Q: Where do I add my image?**
A: Save it to `assets/profile.jpg` and it automatically appears.

**Q: How do I change the colors?**
A: Edit the color values in `css/style.css` under `:root { ... }`

**Q: Can I add more projects?**
A: Yes, copy a project card and fill in your details.

**Q: How do I add my GitHub link?**
A: Find `https://github.com` in `index.html` and replace it.

**Q: Will the contact form send emails?**
A: Currently it's a demo. See [CONTACT_FORM_SETUP.md](CONTACT_FORM_SETUP.md) to enable emails.

**Q: How do I test on mobile?**
A: Open DevTools (F12) → Device Toolbar (Ctrl+Shift+M)

**Q: Can I change the layout?**
A: Yes, modify HTML structure and update CSS accordingly.

**Q: How do I remove a section?**
A: Delete the entire section div and remove its nav link.

**Q: Is this mobile-friendly?**
A: Yes! Fully responsive on all devices.

**Q: Can I use this for multiple portfolios?**
A: Yes! It's perfect as a template.

More questions? Check [TROUBLESHOOTING.md](TROUBLESHOOTING.md) or [README.md](README.md).
